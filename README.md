# Relationship Tracker for RPG Maker MZ.
A relationship management plugin for RPG Maker MZ.

Claude Code assisted in the creation of this program. No creative or expressive assets were generated, just code. Which computers are much better at than I am. But since I can program as well, this is human verified, edited, and debugged. This serves as a note for people who really don't like AI. No judgement at all. I don't like it for a lot of things as well. I just know it's good at patterns, so making math and backend code are great.

View the [tutorial](https://github.com/wg481/relationship-manager-mz/RelationshipTracker_Tutorial.md) for usage.
