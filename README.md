# Relationship Tracker for RPG Maker MZ.
A relationship management plugin for RPG Maker MZ.

This is made with Claude assistance but human debugged, designed, and implemented. I will still plant trees. Sorry.

View the tutorial for usage.

### WG, why?
Paid for plugins kinda miff me. I like free and open source software. So, in the spirit of *Thaler v. Perlmutter*, No. 23-5309 (D.C. Cir. Mar. 18, 2025), *cert. denied* (U.S. Mar. 2, 2026), I insist that there is not nearly enough human authorship, so this code and all things like it are effectively in the public domain. The MIT license is here for people who want to be nice.
